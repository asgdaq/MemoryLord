What's MemoryLord ?

MemoryLord It is a free platform that can be used to facilitate the learning process and to train the memory using flashCards created by you.
Flashcards are effective because they promote active recall in your brain, which is the process by which we retrieve a memory.
Seeing a term and then actively attempting to remember the meaning helps to move it from short-term to long-term memory.

How it works ?

On the main page you will find 2 spaces (word and definition).
In the Word cell enter the new data, and in the second definition.
For example: la maison -> the house



To add this flashcard press the blue button Add.
If you want to remove everything press gray button Clear List
Your flashcard should appear in the table below



Print List if you want to print the table and Start to begin the quiz.
After it , you will select the parameters of the quiz, and pressing Continue the quiz will begin!
If you want to print the quiz , click Print quiz and look down to the screen ,
